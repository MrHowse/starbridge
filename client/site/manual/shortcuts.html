Keyboard Shortcuts

STARBRIDGE v0.08 — QUICK REFERENCE

GLOBAL (All Stations)

KeyAction
F1Open / close help overlay
EscapeClose help overlay, puzzle, crew roster, or cancel mode
CToggle crew roster overlay
[ / ]Zoom out / in (stations with map range control)

CREW ROSTER (C to open)

KeyAction
FCycle filter mode
SCycle sort mode
↑ / ↓Navigate crew list
EnterExpand / collapse selected
EscapeClose roster

CAPTAIN

KeyAction
1 – 6Switch viewport (1 Fore, 2 Aft, 3 Port, 4 Starboard, 5 Quad, 6 Map)
HToggle contact highlights
LToggle contact labels
EnterSubmit captain's log entry

HELM

KeyAction
A / ←Turn left 5°
D / →Turn right 5°
W / ↑Throttle up 5%
S / ↓Throttle down 5%

WEAPONS

KeyAction
← / →Nudge shield focus left / right
↑ / ↓Nudge shield focus up / down
XToggle X-axis lock on shield focus
YToggle Y-axis lock on shield focus
CCentre shield focus (even split)

ENGINEERING

KeyAction
1 – 9Select system by number
DDispatch first idle repair team to selected system
RRecall all active repair teams
BCycle battery mode (standby / charge / discharge)
TabCycle overlay mode

SCIENCE / SENSORS

KeyAction
1EM scan mode (electromagnetic)
2GRAV scan mode (gravitational)
3BIO scan mode (biological)
4SUB scan mode (subspace)

MEDICAL

KeyAction
↑ / ↓Navigate casualty list
AAdmit selected crew member
DDischarge selected crew member
QQuarantine selected crew member
SStabilise worst injury on selected crew
TTreat worst injury on selected crew
0 – 5Filter by deck (0 = show all)

SECURITY

KeyAction
LLock selected room door
UUnlock selected room door
QQuarantine selected room
TabCycle to next marine team

COMMUNICATIONS

KeyAction
No station-specific shortcuts. All interaction is mouse-driven.

FLIGHT OPERATIONS

KeyAction
1 – 9Select drone by index
TabCycle drone selection
LLaunch selected hangar drone
RRecall selected active drone
WToggle waypoint placement mode
PToggle patrol route mode
EnterConfirm patrol route
DToggle decoy deployment mode
BDeploy sensor buoy (survey drone)
EToggle engagement rules (weapons free / hold)
TToggle target designation mode
CClear to land
FFocus map on selected drone
SpaceToggle ship-centred / drone-centred map
EscapeCancel current mode / deselect

ELECTRONIC WARFARE

KeyAction
No station-specific shortcuts. Click targets on map; use buttons for countermeasures and intrusion.

OPERATIONS

KeyAction
No station-specific shortcuts. Click contacts to assess; type advisories in relay input.

HAZARD CONTROL

KeyAction
No station-specific shortcuts. All interaction through canvas and action buttons.

QUARTERMASTER

KeyAction
No station-specific shortcuts. All interaction is mouse-driven.
